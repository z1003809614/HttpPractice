Detailed Description

序列化与反序列化的类， 主要实现了读写int float double, string等类型的变量； 使得后续解析http的内容的时候，能够快速的调用其中的方法，达到内容的快速解析； 设计思路：使用节点链表的形式进行字符内容存储；
